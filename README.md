## Dreams Icon Theme

### Installation
The icon theme can be installed as follows:

```
git clone --depth 1 https://github.com/mi7i/dot-icons.git
cd dot-icons
./install.sh
```

Please note some icons may be missing or named differently. Add this [extension](https://extensions.gnome.org/extension/7397/edit-desktop-files) to verify the icon name in the .desktop file and attempt to locate one from other sources.
